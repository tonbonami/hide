### Short description (한 줄)

### Description
```
HIDE is a top-down stealth game built around a single twist: speed makes noise.

Most stealth games only punish you for being seen. In HIDE, every fast movement
sends out a sound ring — and any guard who hears it turns and comes looking.
So every second is the same decision: rush, or creep?

You have three tools. Walls block line of sight, so you can break a chase by
turning a corner. Shadows make you invisible if you hold still — and they cut
an active pursuit. And the crowd of civilians wandering the room will hide you
if you move among them.

Three gems sit in each room, deliberately placed on the guards' patrol routes.
They're optional. And if you're caught, you lose the ones you were carrying.
Twenty rooms. Sixty gems. Almost nobody gets all of them.

One finger. No tutorial text after the first minute. Reads instantly.
```

### Controls
```
Touch / mouse: hold and drag. The further your finger is from the character,
the faster they move — and the louder they are.
Keyboard: WASD or arrow keys. Hold Shift to sneak.
```

### Tags / genre
```
stealth, action, arcade, single-player, casual, skill
```

### Technical notes (있으면 적기)
```
Single HTML file, ~30 KB. Canvas2D, 60 FPS.
Zero external requests — no CDN, no fonts, no analytics. Audio is synthesized
with WebAudio. 16:9 responsive (960x540 design space). Works in incognito.
Poki SDK not yet integrated — happy to integrate on request.
```
